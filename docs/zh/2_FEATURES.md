# 酷炫功能与核心优势 (PostQ-Tunnel)

## 🛡️ 前所未有的安全层
1. **后量子加密 (X25519MLKEM768):**
   所有数据均在基于量子安全 TLS 1.3 `ML-KEM` 协议上进行传输。有效防止 **"现在窃取，未来量子计算机进行解密" (Store Now, Decrypt Later)** 的降维打击。
2. **无操作系统接管风险:**
   彻底抛弃传统的 OpenSSH 反向隧道方案。不再需要在 VPS 上开启 SSH 用户权限。

## 🚀 惊艳的极速性能
1. **Yamux 数据流多路复用:**
   维持单个加密隧道，即便成千上万人并发访问服务，均被打包在这个专属的高速通道中。
2. **极速内存路由:**
   在服务端采用高并发特性的 Go Memory Map 进行子域动态路由分配。切换时延降至纳秒级别。
3. **绝对零依赖 (Zero-Dependency):**
   无需繁琐部署 Docker，亦无 Python 或 Nginx 依赖。单一静态二进制文件即运行全量服务。

## 👥 面向团队的极简协作方案
1. **IP & CIDR 白名单控制:** 将访问权限限制为仅限企业VPN。防护层位于云端，不消耗本地计算机任何性能！
2. **动态 Basic Auth 认证:** 启动参数携带密码 (`--auth "user:pass"`），防时序侧信道攻击。
3. **智能无人值守重连:** 自带指数退避(Exponential Backoff)休眠补偿技术，断网故障自动无缝恢复。
