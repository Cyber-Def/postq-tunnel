# Круті фічі та переваги (PostQ-Tunnel)

У чому докорінна відмінність **PostQ-Tunnel (qtun)** від класичної зв'язки `Caddy + OpenSSH` або закритих утиліт на кшталт ngrok? Розробляючи цю архітектуру, ми створили проєкт нового покоління.

## 🛡️ Безпрецедентний шар безпеки
1. **Гібридний Post-Quantum Cryptography (X25519MLKEM768):** 
   Трафік між ПК та VPS-шлюзом захищено на рівні TLS 1.3 із квантовим узгодженням ключів `ML-KEM`. Технологія захищає від атаки **«Store Now, Decrypt Later»**.
2. **Нульовий ризик компрометації ОС:**
   Не потрібно створювати bash-користувачів на VPS, як це робить OpenSSH. Агенти працюють через закритий віртуальний шар і не мають доступу до ОС сервера.

## 🚀 Вражаюча продуктивність
1. **Yamux Multiplexing:** 
   Створюється **лише один** постійний тунель. Незалежно від того, 1 чи 10 000 користувачів відкривають ваш сайт, весь їхній трафік пакується у мікропотоки в межах одного TCP з'єднання. Це мінімізує пінги та береже пам'ять.
2. **Миттєвий In-Memory Роутинг:** 
   Жодних bash-скриптів. Проксі-сервер тримає активні тунелі в оперативній пам'яті (Go map).
3. **Абсолютний Zero-Dependency:** 
   Вам непотрібен Docker чи Python. Сервер — це 1 бінарник. Агент — 1 бінарник. 

## 👥 Інструментарій для Команд та Enterprise
1. **IP & CIDR Whitelisting:** 
   Перевірка доступу за IP корпоративного VPN здійснюється на L7 Edge Proxy. Хакери не дістануться до вашого ноутбука.
2. **Dynamic Basic Auth:** 
   Заховайте локальну панель за паролем через прапорець `--auth "user:pass"`.
3. **Ідеально для Unattended (Неспостережуваних) середовищ:**
   Вбудований **Exponential Backoff Reconnect**. При обриві зв'язку агент сам відновить тунель. Система "встановив і забув".
4. **Легка інтеграція SSO:**
   Архітектура Identity-Aware Proxy передбачена для майбутніх оновлень (Вхід через GitHub/Google).
